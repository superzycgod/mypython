# 使用阿里云的源
> 能上互联网的情况下配置使用

## 命令方式
````
pip config set global.index-url https://mirrors.aliyun.com/pypi/simple/
````

## 配置文件

在用户目录下创建文件pip/pip.ini，文件内容如下：

````
[global]
index-url = https://mirrors.aliyun.com/pypi/simple/
````
